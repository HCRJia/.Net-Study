# 1.前端项目创建

1.准备好Node.js和vs code，打开cmd，运行命令：

```plain
npm install -g @vue/cli
```

2.创建文件夹，在文件夹下启动cmd，运行命令：

```plain
vue create workflow_front
```

3.创建好后的文件如下：

![img](https://cdn.nlark.com/yuque/0/2026/png/42432244/1769522008752-6269208a-6cc1-4f3d-8ca3-4cf09e6b1c05.png)

打开VS CODE，打开项目文件夹，打开终端运行：npm run serve

![img](https://cdn.nlark.com/yuque/0/2026/png/42432244/1769522292027-d2aafab0-a08b-4638-be93-68e3f74640c4.png)

显示下面的样子就表示运行成功。

![img](https://cdn.nlark.com/yuque/0/2026/png/42432244/1769522352993-73c42b65-8469-4af6-9da7-ac941ef4f40a.png)
