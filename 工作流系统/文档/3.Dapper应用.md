# 2. 创建模型映射

```plain
[Table("role")]
public class Role
{
    // 主键映射
    [Key, Identity]
    public int Id { get; set; }

    public string RoleName { get; set; }
}

/// <summary>
/// 角色模型-角色表映射
/// </summary>
public sealed class RoleMapper : ClassMapper<Role>
{
    public RoleMapper()
    {
        // 1、映射到ydt_user
        Table("role");

        // 2、自动映射【字段和属性】
        AutoMap();
    }
}
```

# 4. 创建上下文

含义：在Workflow_Back系统中，根据ydt_workflows项目创建上下文。

## 4.1. 创建IWorkflowDbContext.cs上下文接口

在Workflow_Back系统中，创建Contexts目录，并在Contexts目录中，创建IWorkflowDbContext.cs上下文接口，并在IWorkflowDbContext.cs中，继承IDapperDbContext通用上下文接口。并添加IUserRepository属性，创建、继承、添加结果。如下所示：

```plain
using JadeFramework.Dapper.DbContext;
using ydt_workflows_backend.Repositorise;

namespace ydt_workflows_backend.Contexts
{
    /// <summary>
    /// 工作流上下文接口
    /// </summary>
    public interface IWorkflowDbContext : IDapperDbContext
    {
        //1、依赖IUserRepository
        public IUserRepository userRepository { get; }
    }
}
```

## 4.2. 创建WorkflowDbContext.cs上下文实现

在Workflow_Back系统中，Contexts目录中，创建WorkflowDbContext.cs上下文实现。并在WorkflowDbContext.cs中，继承DapperDbContext通用上下文实现, 并实现IWorkflowDbContext上下文接口。并在WorkflowDbContext.cs中，添加数据库选择配置、添加构造函数、实现用户仓储。创建、继承、实现、 添加结果，如下所示：

```plain
using JadeFramework.Dapper.DbContext;
using JadeFramework.Dapper.SqlGenerator;
using MySql.Data.MySqlClient;
using System.Data;
using System.Data.SqlClient;
using ydt_workflows_backend.Repositorise;

namespace ydt_workflows_backend.Contexts
{
    /// <summary>
    /// 工作流上下文实现
    /// </summary>
    public class WorkflowDbContext : DapperDbContext, IWorkflowDbContext
    {
        /// <summary>
        /// 1、数据库选择配置
        /// </summary>
        private readonly SqlGeneratorConfig sqlGeneratorConfig = new SqlGeneratorConfig
        {
            SqlConnector = ESqlConnector.MySQL, // 使用MySQL
            UseQuotationMarks = true // 表和列名使用引号
        };

        public WorkflowDbContext(string ConnectionString) : 
            base(new MySqlConnection(ConnectionString))
        {
        }

        /// <summary>
        /// 实现用户仓储
        /// </summary>
        public IUserRepository userRepository => new UserRepository(Connection, sqlGeneratorConfig);

    }
}
```

## 5.2. 创建UserService.cs Service实现

```plain
public class UserService : IUserService
{
    /// <summary>
    /// 工作流固定类
    /// </summary>
    public WorkflowFixtrue _workflowFixtrue { get; set; }

    public UserService(WorkflowFixtrue workflowFixtrue)
    {
        _workflowFixtrue = workflowFixtrue;
    }

    public async Task<User> GetAsync(int Id)
    {
        //1、查询用户数据
        return await _workflowFixtrue.db.userRepository.FindByIdAsync(Id);
    }

    public async Task<bool> AddAsync(User user)
    {
        return await _workflowFixtrue.db.userRepository.InsertAsync(user);
    }
}
```

# 6. 创建Controller

## 6.1. 创建UserController.cs

在Workflow_Back系统中，Controllers目录中，创建UserController.cs，并在UserController.cs中。实现CommonController 抽象类，同时添加IUserService依赖、Get方法和Post方法。并实现Get方法和Post方法。创建、添加、实现结果。如下所示：

```plain
namespace Workflow_Back.Controllers
{
    /// <summary>
    /// 用户控制器
    /// </summary>
    [Route("[controller]")]
    [ApiController]
    public class UserController : CommonController<UserController>
    {
        /// <summary>
        /// 用户Service
        /// </summary>
        private IUserService _userService;

        public UserController(ILogger<UserController> logger,
                                IUserService userService) :
            base(logger)
        {
            _userService = userService;
        }

        /// <summary>
        /// 查询用户Id
        /// </summary>
        /// <param name="id"></param>
        /// <returns></returns>
        [HttpGet("{id}")]
        public async Task<User> Get(int id)
        {
            // 1、查询用户
            return await _userService.GetAsync(id);
        }

        /// <summary>
        /// 添加用户
        /// </summary>
        /// <param name="value"></param>

        [HttpPost]
        public async Task<bool> Post(User user)
        {
            // 1、添加用户
            return await _userService.AddAsync(user);
        }
    }
}
```
